# Contributing

Thank you for helping improve Agent Integrity OS.

## Principles

- Keep the core deterministic and inspectable.
- Do not replace tests or human review.
- Prefer explicit evidence over inferred confidence.
- Treat restartability as a first-class requirement.

## Development

```bash
npm test
npm run build
npm run benchmark
```

## Pull requests

Please include a `completion-report.json` for meaningful changes.
